First-time setup

  1. Create a new public GitHub repo named staff-scheduler (under your account ssvvsswwii)
  2. In the repo: Settings → Pages → Source → GitHub Actions
  3. Generate a PAT with repo scope and paste it above
  4. Click Save Settings, then add your staff and build the schedule
  5. Click Save to GitHub — the site deploys automatically within ~60 seconds
